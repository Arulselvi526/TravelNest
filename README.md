# TravelNest
A modern and responsive TravelNest-inspired travel website built using HTML and basic CSS. This frontend project includes sections for travel destinations, food recommendations, experiences, and tourist attractions with an attractive user interface and clean layout design. The website is designed to provide a smooth browsing experience while showcasing popular travel spots and activities.

This project helped me improve my skills in HTML structure, CSS styling, responsive layouts, and creating real-world website designs.

# Technologies used
HTML5
CSS3

# Purpose
This project was created to practice frontend web development fundamentals and improve website design and styling skills.
